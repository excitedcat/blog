原来我们一直在追求错误的东西 | Linux篇

ExcitedCat
2014-06-17
linux,windows

做自己喜欢做的事，做自己擅长的事！–巴菲特

从1991年21岁的Linus Torvalds释放第一个Linux版本开始，到现在已经经历了20多年了。从最开始成千上万狂热分子不计得为TA增补、修改、宣扬自由主义精神，到现在名目众多Linux发行版应用于金融、交通、通信、IT等各个核心领域，Linux早已成为颠覆世界的软件，并在以其稳定、高效、安全、免费等特性持续影响着全世界。

我从初中（2008年）开始接触并认识Linux，当时听对面网吧老板说Linux是非常了不起的操作系统，能将磁盘性能发挥到极致。那时候我就问他：你们网吧为什么不安装Linux操作系统呢？

当然这看起来有点像笑话了，不过Linux这个神秘的系统我已经深深记住了。最早开始接触的Linux发行版就是Ubuntu 10.04 LTS了，从那时起我就开始尝试接触各式各样的Linux发行版。当时热衷于将Linux安装到U盘里，随时随地用自己的操作系统查看自己的文档。

CUI & GUI

近年来，随着对计算机领域知识的深入了解。系统学习了一些计算机课程之后开始对OS有了一些系统的理解：

操作系统按照用户界面来分则主要有图形界面和命令行界面，这两种界面没有优劣之分，他们被设计用于不同的领域：

我们熟悉的Window系统采用的界面为GUI(Graphical User Interface)即图形用户界面。我们一打开机器就可以看到有好的图形界面如窗口、按钮、菜单。我们可以方便使用图形化工具去工作，如用PS处理图片、用FoxMail收发邮件、用Chrome浏览器浏览网页。

Linux系统采用的是CUI (Command line User Interface )即命令行用户接口，通过输入命令来管理操作系统。虽然CUI操作方式并没有GUI简单。但是熟练使用可以完成很多GUI无法完成的工作。目前Linux在商业领域大放异彩，其作为服务器操作系统占有市场份额的90%。如今的HTTP Server，Mail Server等多运行Linux操作系统，通常这些服务器只提供服务给消费者，不需要安装图形界面程序。

Windows & Linux 应用领域

从Windows3.0开始到WindowsXP 在到如今的Window7&8，Windows早已用其简单傻瓜的操作方式，易用的图形界面、丰富的应用程序、以及其良好的兼容性颠覆整个世界。现在我们用其消费或创作内容。

Linux最初被设计成为CUI的操作界面，融合了Unix的众多优点，成为极客必备的操作系统。Linux的学习成本很高，TA默认是不提供图形界面给用户使用。这意味着操作系统运行起来后我们面对的就是CUI，也就是命令行。这对普通用户并不友好，而且需要很高的学习成本。同时也不能带给我们好的视听体验。虽然现在有开源的桌面程序加入（Gnome，Xface，KDE），但是缺乏核心应用。

Linux在图形界面的路上走的十分艰难，始终维持着较低的市场占有率，虽然国家在努力号召使用国产Linux操作系统，但是并没有给市场份额带来太大影响。但是Linux凭借其安全、稳定、高效、资源占用小等众多优点，在服务器领域一枝独秀。现在我们用其给用户提供稳定的服务。

如何选择操作系统

对于一个普通的消费内容的用户，选择Windows或MacOS操作系统将会让自己少很多麻烦同时拥有好的体验。如果你需要的是好用的界面、简单的操作那么不要再为选择Linux发行版而劳神费力了。好好用Window或MacOS吧，Linux系统给你提供不了很多好用的程序，更不要提众多的游戏支持了。这也是我折腾操作系统近6年后的经验之谈。

而作为生产着，我们就要选择Linux这样的操作系统，用其稳定出色的性能，源源不断提供服务，供消费者使用。如果你要搭建服务器，最好远离Windows。选择一款稳定且安全的Linux。RedHat，CentOS才是你最好选择。

选择正确的工具做正确的事情，而不是选择错误的东西做正确的事。有时候之所以痛苦，那是因为我们在追求错误的东西。
